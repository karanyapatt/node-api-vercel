# Realtime Project

A real-time application project.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the application:
   ```bash
   npm start
   ```

## Project Structure

- `app.js` - Main application file
- `package.json` - Project dependencies and scripts
- `vercel.json` - Vercel deployment configuration
